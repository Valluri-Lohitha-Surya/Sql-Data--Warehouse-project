SQL Data Warehouse & Analytics Project

Overview

This project demonstrates the design and implementation of a modern Data Warehouse using SQL Server. The solution integrates data from multiple source systems (ERP and CRM), applies data transformation and cleansing techniques, and delivers a business-ready analytical layer for reporting and decision-making.

The project follows the Medallion Architecture approach, consisting of Bronze, Silver, and Gold layers, to ensure scalable and maintainable data processing.
